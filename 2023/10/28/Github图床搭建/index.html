Github图床搭建
2024-04-21 14:34:05 # 教程 #Github #图床

新建 GitHub 仓库

  1. 登录/注册 GitHub
  2. 新建一个仓库，填写好仓库名
  3. 仓库描述
  4. 将权限设置成 public

生成一个 Token

在设置-developer setting里面

只需要勾选 repo

配置 PicGo

前往下载 PicGo（点击下载），安装好后开始配置图床

  1. 设定仓库名：按照 用户名/图床仓库名 的格式填写
  2. 设定分支名：main
  3. 设定 Token：粘贴之前生成的 Token
  4. 指定存储路径：填写想要储存的路径，如 img/，这样就会在仓库下创建一个名为 img 的文件夹，图片将会储存在此文件夹中

CDN

jsdelivr

设定自定义域名：它的的作用是，在图片上传后，PicGo 会按照自定义域名+上传的图片名的方式生成访问链接，放到粘贴板上，因为我们要使用 jsDelivr 加速访问，所以可以设置为https://cdn.jsdelivr.net/gh/用户名/图床仓库名

githubproxy

https://github加速URL/https://raw.githubusercontent.com/用户名/仓库名/main/

注意：

GitHub图床仓库大小不能超过1G。当仓库超过1G后会有人工审核仓库内容，如果发现用来做图床，轻则删库重则封号。

建议到500M就新建一个仓库
